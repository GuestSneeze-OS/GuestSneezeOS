# GuestSneezeOS
## Features
* Any computer with an AMD or Intel GPU & CPU can run GuestSneezeOS
* Supportability
## Software preinstalled
* Steam (GSOS 1)
* Firefox (GSOS 2)
* Discord (GSOS 9)
* Dolphin (GSOS 1)
* Waydroid (GSOS 9)
* Lutris (GSOS 6)
* GameMode (GSOS 9)
* Gamescope (GSOS 1)
